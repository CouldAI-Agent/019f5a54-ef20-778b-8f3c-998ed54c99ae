# The World Wide Web - Presentation App

A beautiful, interactive 20-slide presentation built with Flutter, covering the foundational concepts of the World Wide Web.

## Overview
This application serves as an educational presentation covering:
- Overview of the World Wide Web
- The Distributed Web
- The Addressable Web (URLs and URIs)
- The Linked Web (Hypertext and Graphs)
- The Protocols of the Web (HTTP, TCP/IP, DNS)
- The Searchable Web (Crawling and Indexing)
- The Languages of the Web (HTML, CSS, JavaScript, WebAssembly, Backend)

## Features
- **Interactive Slides**: 20 beautifully formatted slides with rich background imagery.
- **Responsive Layout**: Adjusts layout automatically. Desktop/Web displays side-by-side text and imagery, while mobile/embedded devices use a full-screen blended background to preserve screen real estate.
- **Cross-Platform**: Built for Web, Mobile, and Desktop using Flutter.

## Setup Instructions
To run this project locally:

1. Ensure you have the Flutter SDK installed.
2. Clone the repository and navigate to the root directory.
3. Fetch dependencies: `flutter pub get`
4. Run the app: `flutter run`
